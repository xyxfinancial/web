    记：思维模型 - 梳理1¶

    起因¶

    一直都收集各种思维模型，
想应用于自己的生活，
但是没有认真的梳理思考理解和应用过。

那就先初步做一遍吧。

    时间管理¶

    # 四象限
最好只做重要不紧急的事
通过计划管理，减少重要紧急的事发生

# 80/20原则
做减法，排优先级，不怕主动的事情多，事情多才有选择和需要做选择

# GTD
收集，整理，组织，回顾，执行

# ZTD
收集-处理-计划-行动-清单-整理-回顾-简化-习惯-热爱

# 柳比歇夫时间记录
toggl记录

# 番茄钟
Forest，习惯开声音的话，会帮助集中注意力

    清单¶

    "关键点"比"大而全"重要
能使用团队力量，权力下放
简单至上，持续改善
# 设定检查点
关键词-扩展层次-再补充成具体的句子 (可按列扩展)

# 分类(操作确认，边读边做)
执行清单，核查清单，核查沟通清单

不要太长(5-9点)，用词准确，注重版式，现实检验

    AFD清单¶

    action，focused，dessert

1. 细分/拆分任务到可执行
具体到数量，目的，关键词
todo：怎么分

2. 按所需的不同思维方式/能力建清单 - 分段工作法 (莫法特工作法)
想象力
逻辑分析，推理，归纳力
阅读理解力
数学计算力
乐感，美感
空间几何感

3. 成就感清单 - 简单，无需动脑，没有时间限制的小事
整理桌面
整理电脑文件夹
整理notion
整理照片
看视频，听讲座
(搜索图片，准备图片)

    精力管理¶

    体能-情绪-注意力-意义感

    内容多，作为明天的主题

    精要主义¶

    "我选择做"，而不是"我必须做"
不加思考的努力 = 习得性无助
认为几乎一切都不重要，区分重要的少数和不重要的多数
拒绝好机会，拥抱真正的契机；只接受10%的机会

# 留出空间
设计人生，专注探究，阅读思考

# 审视查找
信息精华，相关联，大局观

# 重视游戏
激发创造力

# 重实质而不是形式，具体的目标，制定自己的规则

# 放弃+止损
- 去避免
禀赋效应(拥有即高估)
对浪费的恐惧
强己所难(寻找中性意见)
现状偏见(因在做而继续)
随意承诺，害怕错过

# 设置缓冲
比预计多留50%

# 扫除障碍
确定目标(扫除什么)，从小处开始

# 关注小胜
最低可行性进步(完成比完美重要)

# 心流
设计常规，习惯=刺激+常规+激励，先啃最难的骨头，混合型常规

# 专注
想清楚当下最重要的，清单排列优先顺序

    采铜法则¶

    # 对待时间
工作要快生活要慢，提升时间使用深度(找到并保持一项长期爱好)

# 做更好的选择
以人生最高目标为第一原则，发现人生更多可能性

# 即刻行动
现在就是最好的时机
从工作核心区开始
三行而后思(信息，预期，结果，进度，工具，情绪，阻碍，优势，遗憾，意义)
三个关键：保证及时性-梳理反应链-关注意外

# 高段位学习
向自己提问
三个层次：信息/知识-价值-形式/构成
三条路径：不只寻路径还要寻过程-不知做归纳还要做延展-不只比较相似还要比较不同
