    樊登读书¶

    康妮《如何结交比你更优秀的人》¶

    康妮，哈佛MBA，人脉管理师

    向上社交 Network up¶

    # 周围所有的人身上都有他的优点和优势，是你不具备的
1. 去识别他们身上的优势和优点，去向他们学习
2. 把他们的优势和优点变成你自己的
3. 跟他们进行有效的链接

# 从陌生人变成熟人，把熟人变成朋友，知己，智囊

    人脉力¶

    # 识别力
能从身边的人身上识别优点，亮点，特色，强项

# 学习力
学习更优秀人身上的优势优点，能把它们变成自己的

# 链接力
能跟优秀的人产生有效的链接

    人脉搭建的误区¶

    # 不愿，不敢，不会
    不愿¶
    没有用开放的心态
真心认可一件事物时，会有推荐分享的冲动
点燃好奇心
真正了解对方，看到亮点特点
    不敢¶
    # 内向
社交觉得疲惫

1. 制定一个合理的社交计划
跟一个人，两个人建立一个深层次的联系

2. 着重建立一对一的关系
一对一的关系是内向人的特长，
在这样的关系中，可以深入地跟进行聆听，沟通，谈话
这种关系是一种长远的，深厚的关系

3. 把自己变成某个领域的专家
创造演讲和发言的机会
成为场合的中心
发光的个人
不需要主动社交
变主动为被动

4. 动静结合，别让自己透支
一个星期做一次午餐活动
一个月有一次双人聚会
一个季度组织一个十人聚会

5. 舒适区外面不是恐慌区
在舒适区和恐慌区之间有一个巨大的圈 -> 学习区
扩展自己，具备更多能力

# 怕拒绝

认为普通人，没特点，没优势，也没有价值

1. 增强自我认知能力
每个人都有价值
挖掘自己的优势亮点
有激情的东西 -> 能给他人提供价值

有兴趣，有激情 -> 产生心流
有心流 -> 多学多问，多看书，多向他人学习
成为专家 -> 价值

2. 拥有成长型心态
告诉自己not yet
现在没做到，将来可以做到
eg. 人脉搭建，没有共同话题，不见得下一次也没有话题
做准备
eg. 见陌生人，了解背景，工作经历，社团等

3. 写出恐惧清单
恐惧是内心写照，不是真实情况
高能量的姿势 (哈佛大学社会心理学家艾米·卡迪)
    两分钟：两脚与肩同宽，挺胸抬头站起来，两臂向上伸展，像超人一样
冥想正面练习
巩固本源，自我认知，更放松的状态
    不会¶
    与人交往，不是一腔热情
人和人交往，有一个度

对对方没有足够了解时，
一厢情愿把自己认为好的东西推销给别人，
在对方看来是一个不得体的行为

学习与人接触，既有热情，有热度，有温度，还能得体

# 行动得体的原则
真诚
有感恩的心
大方
谦虚
自信

    人脉搭建的“道”¶

    # 保持好奇心
对世界，对他人真正感兴趣，
渴望通过人和人的交往，看到更广阔的世界，
通过和他人的深入了解，认识他们的本真
对话的重点：感兴趣，试图了解

人际关系大师戴尔·卡耐基博士说，
每个人内心深处都是要得到别人的认可，要感觉到自己的重要，
当你对一个人感兴趣，聊他感兴趣的话题，
重点放在去了解他这个人的时候，聊天就不会进行不下去

真的好奇是当你问了一个问题以后，要根据他的答案，
根据他跟你讲述的内容，再提出针对的好奇的问题

# 真诚温暖
真诚温暖是敲门砖，
真诚用言语用行动传达给对方，
让对方感受到你的光和温度，被你吸引

真诚温暖是两个不同的概念
    真诚指你的发心
    我跟你结交是我真正地想去了解你
    温暖是你的行动
    通过话语，笑容，整个肢体语言传达

eg. 眼睛看着对方，充满笑容，语速可能会快，声调可能会高，身体可能会整个前倾，甚至不自觉地加入很多手势的动作

真正的得体是建立在真诚待人的基础之上，
真诚就是真实和诚意表达出来，
让别人感受到就是和别人相处的第一步

诚意是以互利互惠为本
真诚地希望跟对方交好，
真诚地希望能够给对方提供价值，
通过这个关系能够使对方受益，
同时自己也能有所增益

# 利他心态
一个好的关系，要从为他人创造价值开始
任何从小的一点出发，只要能想到帮到对方的点都是价值

    人脉搭建的“术”¶

    # 把陌生人变成熟人
1. 赞美
通过跟别人的任何一次交流和交往，把心里的善意传达
eg. 当观察一个人的头15秒钟，所有的东西都可以赞美

# 把熟人变成朋友
1. 朋友是麻烦出来的

主动请求别人帮一些小忙，有利于信任的养成和关系的深入
请别人帮忙，要只是偶尔，是对方有能力做到的，也有一个度的问题
2. 展现自己真实的一面，适当暴露柔软和脆弱

# 好心态
每个人都有自己的优点和缺点，
如何利用自己已有的东西，仅有的价值提供给对方，
同样如何从对方身上学习到没有的东西，自我提升

# 生活中结识"大咖"
从赞美开始 -> 找到共同点 -> 持续互动 -> 给对方提供价值
充分准备 -> 了解信息 -> 自我介绍 -> 链接时展露闪光点
阶梯型社交 -> 从身边人，团队成员开始 -> 证明靠谱 -> 才可能介绍
不气馁，复盘总结 -> 十个大咖，有一个链接就很好
积极主动 -> 向上社交不是一日之功 -> 一点一点积累
