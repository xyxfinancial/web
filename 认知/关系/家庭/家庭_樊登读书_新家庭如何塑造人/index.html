    樊登讲论语¶

    子贡欲去告朔之饩羊。子曰：“赐也，尔爱其羊，我爱其礼！”

    推测是发生在孔子晚年的时候

子贡作为儒商鼻祖
富有程度甚至能够与君王分庭抗礼
他到任何一个地方
国君都要给他让出半个席子请他入座

## “子贡欲去告朔之饩羊”
子贡很有钱，又是个外交家
他承担了为告朔祭祀提供羊的责任
但可能觉得有点心疼
就说，每个月杀一只羊太浪费了
要不别祭祀了吧

- “告朔”
一种祭礼，每个月的初一叫作“朔”

- 饩（xì）
羊

## “赐也，尔爱其羊，我爱其礼！”
子贡啊，你爱的是那头羊，我爱的是那个礼

- 赐
子贡的名

孔子的这话是他发出的一个感慨
感受到了历史进步的车轮，谁都无法阻挡
生活中本就存在着大量的争论
处理一个又一个的矛盾
时代在进步
新的矛盾也在不断地产生
社会本来就是在矛盾中发展

痛苦是随着能力的改变而改变
能力不是做事的能力
而是体会痛苦和幸福的能力

如果一个人体会幸福的能力比过去高了
幸福的总量就会增加

    # 总结
孔子，孟子，王阳明都教大家修炼

- 修炼的目的
不断地提高自己感受幸福的能力
孔子总是随时随地能够高兴
能够以积极的态度看待外界的变化

    Gallup 盖洛普¶

    Influencing - Command 影响力 - 统率¶

    # 特质
1. 命令引导负责
2. 不会因将自己的观点强加于他人而感到不适
3. 一旦意见形成，需要与他人分享
4. 一旦设定目标，就会感到不安，直到别人保持一致
5. 不怕对抗，相信对抗是解决问题的第一步
6. 需要让人与人之间的事情变得清晰
并挑战大家保持清醒和诚实
7. 推动冒险，虽然有人不满，贴上固执己见的标签，但通常会心甘情愿地听从指挥
8. 存在感和指挥权，采取立场并要求朝某个方向移动

# 自我行动
1. 寻找需要说服他人的角色，eg. 销售
2. 随时准备对持，练习语气和技巧
将对持能力转化为真正的说服力
3. 抓住机会坦率而直接地谈论敏感话题
不隐瞒真相，成为一个坦率的人
4. 帮助他人做出承诺，激发行动的火花
5. 找到相信并支持的事业
6. 征求意见，解释坦率不是要吓唬保持沉默

# 适合合作或发展的优势
Woo 取悦
Empathy 体谅

# 如何与之合作
1. 当需要一个项目重新开始
当需要人们被说服时，请让他负责
2. 请他评估组织中正在发生的事情
最有可能给一个直接而不同的答案，想法
3. 尽可能多地给他领导和做出决定的空间
不喜欢被严密监督
4. 除非100%准备好坚持到底，否则不要威胁
5. 帮助他人了解他的自信是使有效的一部分

    樊登读书¶

    新家庭如何塑造人¶

    《The new people making》
萨提亚的奠基之作

# 成为内心和谐的人
唤醒个人内在价值

# 家庭系统的四方面
1. 自我价值感
一个人对自己的感觉和想法
2. 沟通
人们之间传递信息的方式
3. 家庭系统
家庭成员遵循着应该如何去感受和行动的规则
每家有不同的规则和潜规则，独特的家庭系统
4. 家与社会的联系
与其他人及家庭外的组织发生关系的方式

- 在每个家庭里，成人的责任很重
扮演着塑造者的角色
构造家，发展环境特点
让家能够健康，能够理得顺

# 存在问题的家庭
1. 自我价值感低
家成员常发脾气，觉得被忽略，被误解
2. 沟通间接，含糊，不真诚
说的东西不清楚
互相不明白对方意思
不愿意去尊重
3. 规则严格，非人性化且不可谈判
像筑起钢铁牢笼，上面还可能带刺
4. 谈到外部世界是担心，紧张，防范的

# 健康的家庭
1. 自我价值感高
2. 沟通直接，清楚，明确，真诚
3. 规则富有弹性，人性化，恰当而且可变
4. 与社会的联系是开放，充满希望的
是在选择的基础上建立

# 对家庭评估的问题
1. 对现在的家庭生活感到满意吗？
2. 觉得家人似朋友般亲切，彼此爱护，相互信任吗？
3. 作为家庭一员，认为是一件令人愉悦，令人兴奋的事吗？

- 改进

    自我价值¶

    # 问内心满不满
有没有自尊
觉得状态是不是充满能量
是不是满意
觉得自己是不是有价值感的人
是不是被爱

# 被爱的人都对改变持开放的态度
当人不觉得自己被爱时，会觉得自己被挑战，被挑剔，被要求，被瞧不上，是不愿意做任何改变的

# 给孩子高自尊水平
说话是鼓励的
包含爱，好奇心，探索，包容，以他为荣
让孩子理解家庭里发生的事

# 每个人的指纹都是不一样的
这种独特性决定每个人自身拥有的价值
“天不生人上之人，亦不造人下之人”
从根本上理解自己的独特性和价值
不需要依赖于别人的肯定
当具备了这种能力
反而会获得更多他人给的爱

# 拥有自信时的状态和表现
当自我感觉良好，自我欣赏时
会以一种高贵，真诚，勇敢的姿态
充满活力和爱心地来应对生活

# 爱自己是认可自我价值的一种表现
一个高自尊水平的人所具有的感受力
在为外在付出之前，先爱自己
为他人付出也是爱自己的一个表现
付出得越多，得到的越多
不需要再去追求别的东西
因为在付出的那一刻
其实就已经得到了

    沟通，交流¶

    # 沟通绝不仅仅是通过语言
是带着身体，价值观，经验，感官，语言，头脑，情感一起
是一整个综合的系统在沟通

- 和小孩说话，最好蹲下来
能够目光平视

- 家里边不能传话
本来不是是非的事
语气只要一调整，意思就可以完全改变

- 模仿对方的语气
感受所说话的本质含义

# 沟通陷阱
- “暗示法”
不说，希望对方能自己明白

- “读心法”
各种各样自己轻易做出结论会使家庭矛盾重重

# 四种错误处理压力的模式
1. “讨好型”
内心独白是“觉得自己什么都不是，没有价值”
是一种逃避，在逃避沟通所带来的压力
来自儿时教育，父母要求别影响别人，说为了自己而向别人提出要求是自私的

2. “指责型”
内心独白是“觉得孤独而失败”
心中孤独和失败，觉得谁都不能指望，靠不住
来自儿时教育，父母说，不要让任何人拒绝，别懦弱，必须去争取，要更强悍

3. “超理智型”
内心独白是“我感觉很脆弱”
无论别人出现什么情绪，发生什么状况，都没有情绪上的反应，以超理智的方式对待
来自儿时教育，父母强调聪明，精明

4. “打岔型”
内心独白是“没有人关心，没有自己的位置，没有自己的空间，在这完全不重要”
来自儿时教育，父母说，别太认真，谁会在意呢，没关系”，等玩世不恭的态度

# “心口不一”
想要表达孤独，
想要表达无助，表达的是愤怒
想要表达和解，
想要表达贡献，表达的是讨好

大脑预先加工了很多可能性
受到之前所带来强大可能性的背景影响
不敢说实话，不敢直面事实

# “一致型的表达”
内在，外在，所说的语言，身体的姿态，传达的信息是一样的
不要自己吓自己

    家庭系统¶

    # 要有真正的私人空间
越是有真正的私人空间
越是尊重个人的私人空间
家庭系统越是开放

# 运作的系统
1. 目的或目标，家庭的蓝图
家庭要往哪儿去
2. 基本的组成成分
家里的成人，孩子，男人，女人
3. 各部分工作的次序
4. 系统中供给能量以使各部分运转的能力
不光是食物
还包括爱
包括知识
包括与外界相互作用的方式

- 封闭和开放
主要区别在于对变化的反应
对变化的反应决定着系统的健康程度

- 封闭系统
拒绝一切变化
觉得所有的变化都会带来挑战
基本信念认为，人们从本质上来说是邪恶的
必须持续受到控制才能表现得好
关系必须通过强制的力量或者对惩罚的恐惧而得到规范
相信最有权力的人拥有正确的方法，总有人知道什么是最好的
在封闭的系统当中，自我价值次于权力和成就

- 开放系统
自我价值是首要的，权力和成就是次要的
第一每个人是不是在成长
第二行动代表了一个人的信念
变化受到欢迎，被视为是正常和可取的
交流，系统和规则都彼此联系
系统中最重要的事情不是奉献，而是成长
每个人在系统当中都在不断地成长，得到

# 拥有高自尊
1. 没有两个人是完全相同的
每个人都是独特的
2. 没有两个人的时间表是完全一样的
即使行事的方式是相似的
3. 周期性的孤独是身为一个独立人的自然结果
人不会因为孤独而死去
不要被恐惧所裹挟，好多人过度地索取要求，是来自恐惧孤独，觉得不能一个人
4. 要能够接受一个人的状态
去享受两个人在一起的互动
做好的家庭建筑师
家需要建设，而不仅仅靠爱就足够

# 家庭有很多关系，最基本的三角形关系
整个家是由大量的三角关系构成

# 一生家庭生长规律，10次危机
1. 受孕，怀孕和第一个孩子出生
2. 随着孩子开始使用智慧的语言
3. 孩子和家庭外部世界产生正式的联系
上学的时候
4. 当孩子进入青春期时
5. 当孩子长大成人，并离家寻求独立时
6. 当年轻人结婚时
配偶将作为一个陌生人的元素被家庭所接受
7. 女性绝经期到来时
8. 更年期
9. 成为祖父母
10. 当死亡降临到配偶中的一方
然后是另一方

- 整个家庭像一个生物
有自然的生物节律
坦然地面对和接受
最核心的方法是保持每个人的高自尊水平
以及充分地沟通

# 家庭蓝图
不需要特别具体的蓝图
人和人是完全不一样的

- 欢乐可以滋养个人成长
好的家庭相信欢声笑语
快乐的表情在家庭中非常重要

一个人会学习来自多巴胺的刺激
多巴胺的刺激就是来自快乐

- 阻止实现蓝图的东西
1. 首先是无知
不知道什么叫家庭系统
不知道什么叫自尊体系
不知道沟通的重要性

2. 无效的沟通

3. 不确定的价值观
父母跟孩子沟通的东西都来自实用主义
父母没有稳定的价值观

- 好的蓝图
核心是发展
家庭个体的自尊水平都高
希望人和人的关系和谐
一个人对另一个的反应是有爱，开心喜悦的
希望对外界资源能够更好地了解和使用
调整好对外部世界的看法
及对死亡的看法

- 成长型的蓝图
不是具体简单的目标
蓝图是来自心灵的成长
来自人和人之间关系的改善

- 当延伸家庭变得更大
认真地对待家庭里的每一个成员，特别是老人
关注每一个人独有的特点，个性鲜活
不要贴标签，刻板的称呼，角色，脸谱化

- 真正的二人关系建立在独立性上
要允许对方成为自己

# 激发孩子的灵性，生命力
人有想法，创造力，审美能力
提供尽可能丰富的环境
多看，多体会，多做些解释
