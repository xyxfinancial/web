    樊登讲论语¶

    子谓子贡曰：“女与回也孰愈？”对曰：“赐也何敢望回？回也闻一以知十，赐也闻一以知二。”子曰：“弗如也；吾与女弗如也。”

    # 女与回也孰愈
孔子就问子贡：“你觉得你和颜回谁厉害？”

# “赐也何敢望回
子贡说：“我哪敢跟颜回比啊。”

# 回也闻一以知十，赐也闻一以知二
子贡说：“颜回是闻一知十，我是闻一知二，我俩差远了。”

# “弗如也；吾与女弗如也。”
孔子接着又感慨：“确实不如他。”

    # 总结
孔子是在思念颜回
在强烈的思念之情下
孔子说，颜回的境界真的很高，其实咱俩都不如他

    樊登读书¶

    解惑：心智模式决定你的一生¶

    二十世纪六十年代非常著名的思想家
E.F.舒马赫

# 《活出生命的意义》

# 《高效能人士的七个习惯》

    # 思维地图

# 世界最少有四个层次
1. 矿物质、无机物
不动，没有生命，可以研究分子结构的东西
2. 植物
有生命，会生长，会吸取营养，发生光合作用
3. 动物
有意识
4. 人类
自我反思

最本质的区别在于有一个从被动到主动的过程
动物的主动全部来自于动机
人有意志，意志就是人接近神的地方

- 认知模式向人的一端进化
人是有自由意志，可以自己做出选择
唤醒内心中的选择权，发挥聪明才智和主动性

每个人都有获得自由的可能，选择决定此刻自由不自由

# 一体化的过程
矿物质能做到完全割裂，人做不到
人需要追求内在的整合同化，割裂就会痛苦

# 存在层次将决定生活状态
假如对这个世界的看法是错，就会陷入伯特兰·罗素所说的“不屈不挠的绝望”

# 两个契合
1. “头脑和心智的契合”
思考所有的问题不能够都只通过理性来分析
只通过理性就是头脑，用头脑来代表理性
感官只能带来资料，比如说眼、耳、鼻、舌、身
这个资料最后会变成什么东西，完全取决于心智

在对待比无生命物质重要性等级更高
或存在层次更高的事物时
观察者不光要依赖本人通过学习和训练培养的更高才能
还要依赖信念的契合
也就是基本预设和基本前提的契合

不是简单地吸收外部的信息来分析
而是心中得有一些信仰，一些相信
才能够把它们整合在一起

不能够只相信外部的数据
们需要有内在的信仰，原则，道德

- 心智之光

2. 理解的科学与操纵的科学的契合
思想家帕斯卡尔，“心有其种种理由，而理性对这些理由一无所知。”

如果不能够把一个东西用指针读数表示出来的时候
就不能够进入教材成为公共知识

- 操纵的科学
可被使用，证明

- 理解的科学
智慧

当操纵的科学服膺于智慧
也就是理解的科学时
科学就是最有价值的工具
不会造成任何损害

- 不要做"手段上的富人，目的上的穷人"

# 四种认知领域
1. “我-我自己的内在” (自知者明)
我的内心世界正在发生着什么
这个事只有我自己知道
但是很多人自己不知道
因为很多人根本不看自己的内心
他自己的内心世界究竟发生着什么
大部分的人是不了解的

- 英雄之旅
一个英雄的成长的过程
不是他征服外在世界的过程
一定是他完成了一个内心之旅
跟自己达成了一个最后的和解
这个才是英雄

每一个人都有机会去了解自己的内心
完善自己的内心
解决内心当中隐秘的问题

- 调动出自己裸露的注意力
完全投入地在关注自己的念头

念头的好坏不是问题
真理、真实、上帝、涅槃是不会被思绪找到的
因为思绪属于意识确立的存在层次
而不是自我意识确立的更高层次
在后一层次，也就是自我意识确立的这个层次
思绪有其合理的地位，不过是从属地位
思绪不可能导向‘觉醒’
因为关键就是要从思考中觉醒过来，进入寻求之中
思绪可以提出诸多问题
它们或许都很有趣
但它们的答案并未让我们醒觉
它们被称作‘妄念’

对于思想的观察，就是自我反思
如果一个人整天想追求自己的内心
想知道自己内心到底在想着什么
想做内观的事情
在这个社会会显得很另类

想要觉醒就意味着要脱离“计算机”
成为“编程人员”
得不那么惯性地生活
而应该站出来观察自己，觉知

2. “你-你的内在”
其他存在的内心世界正在发生着什么
你们心里正在想着什么

- 认识别人的想法是一件极其困难的事
a. 说话者必须准确地知道想传达的想法
b. 必须找到可见的手段：姿势、身体活动、言辞、语调将‘内在’的想法‘外化’
c. 听者必须准确无误地接受这些可见的手段
d. 听者必须以某种方式将接收到的多种沟通手段合而为一，把它们变为思维

人想要识别别人的想法
本身是一件非常难的事
除非是用自己的感觉
用自己的内心在交流

商业的能力来自于对别人的感知力
“推己及人”
在于怎么了解自己的想法，怎么观察自己
而不是惯性地生活

3. “我-我的外在”
我在其他存在的眼里是什么样的
你们会怎么看我
我在你们心中的形象是什么样子

- 了解别人对自己的认识
a. 意图与行动到底哪一个更重要
判断自己都是从意图的角度出发
别人看不到意图，别人看到的是行动

b. 意图和行动到底哪一个更真实
在自己看来，可能觉得意图是最真实的，所以委屈
在别人看来，说的都没用，只看行动，所以行动更真实

考虑自己的问题都是从意图出发
看待别人的事情都是从行动出发
造成了对自己的评价和对别人的评价完全不一样

4. “你-你的外在” (知人者智)
我所看到的周遭世界是什么样的
我不知道你们心里想什么
但是我看到的这一切我能够知道
我能描述出来

- 对外部世界的认识
不能够简单地用物理学和数学来指导人类的思想
想更了解外部世界，不仅靠科学，还应该靠整体的思想

进化现象

四个认知领域的统一性
只研究外在现象，只偏重于其中一种
会带来统一性的破坏
人类社会，现在都偏重于第四种
都在看外部世界，在想办法了解外部世界
但很少了解自己的内心
也很少了解自己在别人眼中的形象
也不知道别人内心在想什么

# 更高层次的心智模式
1. 汇聚性问题
可以找到一个统一的方法来解决的问题
所有用科学的手段能够解决的，化学、物理学能够解决

2. 发散性问题
在发散性问题上逻辑帮不上忙
答案一定来自于更高的层次

eg. 自由和平等是矛盾的
但当加入“博爱”词后，自由和平等可以和谐相处

管理者的认知，减少团队的矛盾
唯一的办法是要唤醒每一个人的内在成长的能力

发散性问题没有解决
而且最重要的是
发散性问题一旦解决，就会变得死气沉沉
这就是生命所具备的痛苦，生命所具备的纠结

# 总结
人生就是一连串不可避免会遇到的
并且必须以某种方式来解决的发散性问题
只靠逻辑和推论的理性很难驾
可以说
这些问题充当了一种拉伸器具
用以培养完整的人
这意味着发展的是人超越逻辑的能力

生活就是道场
此时正是修炼时
人生所面对的所有的纠结、痛苦、烦恼
都是在不断地拉伸我们
然后试图让我们跳到另外一个层次来看待这些问题
如果不能够让自己变得不一样
不能够跃迁
就永远会跟那些问题纠缠在一起
会觉得极其痛苦，没法解决

人和矿物质不一样
有选择权、有创造力、有好奇心
所以可以想办法改变自己、提升自己
改变自己生活的方式和习惯
