    复杂时代的不变算法¶

    利用历史周期应对未来发展¶

    影响周期的三大因素¶

    1.大量债务的创造，以及为应对这些债务而大量印钞
2.与1相关，巨大的贫富差距和政治鸿沟出现，特别是在欧美
3.大国崛起，挑战已有的大国和已有的世界秩序

    公元1500年以来的各个帝国的演变¶

    # 11个主要帝国的发展弧线
红线表示的是中国
中国是公元1500年前后最富有，最强大的国家
在1840年左右衰落，所谓的"百年屈辱"

# 1949后中国强劲崛起的三个阶段
1949-1976年，奠基阶段
1978-2008年，第二阶段，透过改革开放赢得实力，同时未对其他国家造成威胁
2008年至今，第三阶段，巩固之前的成就，推动中国前进；中美冲突日益凸显，全球化走向终结

    1978年至2008年中国快速发展的原因¶

    # 世界仍处于大周期里的和平与繁荣时期
全球化和市场经济是广泛被接受通往更美好世界的道路
- 信念
商品和服务应当在成本最低的地方生产
应当让人才自由流动，不因国籍受歧视
淡化民族主义
重视全球机会均等和寻求利润的市场经济

# 改革计划经济，实行改革开放政策
取得了很好的效果
中国学到了很多东西，吸引了很多外资，成为一个巨大的出口国和储蓄国

# 2012年中国新一届领导集体诞生
政府积极推行改革政策，同时收紧了其他管制措施
开放市场和经济
管控债务增长
提高人民币汇率的灵活性
支持创业和以市场为导向的决策(特别是那些中国希望领先世界的产业)
建立合理的监管体制，由高度完善的监管机构实施监管
发展未来科技和工业潜能，扩大经济效益，使其惠及最落后的群体和地区
控制污染和环境退化

    国家兴衰三阶段¶

    # 上升阶段
国家建设的繁荣时期
在一个新秩序建立之后到来
(新秩序通常出现在战争之后)
国家根基强大
因债务水平较低，国民的财富差距，价值观差距和政治鸿沟较小
人们有效地合作，创造繁荣
教育和基础设施良好。领导者强大而有能力
世界秩序和平有序，由一个或多个主导性的世界大国引导

# 顶部阶段
高负债
财富、价值观差距和政治鸿沟巨大
教育和基础设施的质量下降
国家内部不同群体的民众之间存在冲突
由于扩张过度的帝国受到新兴竞争对手的挑战，国家之间发生争斗

# 下跌阶段
争斗和重组的时期
在新的内部和外部秩序建立的过程中，出现冲突和变化
债务
内战和革命，国外战争
人们对货币失去信心

    重要的力量¶

    # 自然现象
流行病，干旱和洪水，全球变暖等气候变化

# 人的创造力
人的创造力和适应性，使生活水平逐渐大幅上升

    五大类型战争¶

    贸易/经济战
技术战
地缘政治战
资本战
军事战

其它：文化战和自我交战

    多代人心理周期5阶段¶

    # 早期新兴国家
人民和他们的国家都很贫困
人民认为自己贫困
国家的发展速度取决于它的文化和能力
这个阶段的国家称为""

# 后期新兴国家
人民和他们的国家都富裕了
人民仍然认为自己贫困，仍然努力工作，大量储蓄
投资于人力资本开发，基础设施建设，研发等
一个大国走过这个阶段后，通常会成为世界强国

# 健康高峰国家
人民和他们的国家都富有
人民认为自己富有
优先重点从强调努力工作和储蓄以免受到困难时期的影响，转为品味生活中更精美的东西
更愿意多花钱，艺术和科学通常得到蓬勃发展
美国在1950-1965年处于这个阶段，中国正在进入这一阶段

# 早期衰落国家
人民和他们的国家变贫困了
人民仍然认为自己富有
债务相对于收入上升，劳动者的收入和支出都很高
由于基础设施、资本品和研发方面的投资效率下降，城市和基础设施变得陈旧和低效
国家越来越依靠自己的声誉(而不是竞争力)来为赤字融资

# 明显衰落国家
人民和他们的国家都贫困
人民认为自己贫困

    # 大周期
经济/金融地位：债务负担；预期增长率
内部秩序：贫富，机会差距与价值观差异；内部冲突
外部秩序

贸易

创新与技术
成本竞争力：根据质量和生产率因素调整后的劳动力成本
军事实力
市场与金融中心地位
储备货币地位

# 其它指标
基础设施与投资
个性/文明/决心
地质
资源配置的效率
治理/法治
自然灾害

# 截至2021年8月，11个主要经济体的分数
美国0.87、中国0.75、欧盟0.55、德国0.37、日本0.30、印度0.27、英国0.27、法国0.25、荷兰0.25、俄罗斯0.23、西班牙0.20

    美国¶

    # 大周期看来不利
债务负担高
未来10年的预期实际增长率相对较低(每年1.1%)
内部混乱风险大：美国收入最高的1%和10%的人分别占有19%和45%的收入
外部混乱风险存在：美国和中国冲突

    中国¶

    # 大周期看来较有利
债务负担低(且大部分债务以本币计价，风险更低)
未来10年的预期实际增长率相对较高(每年4.3%)
内部混乱风险适度：中国收入最高的1%和10%的人分别占有14%和41%的收入
外部混乱风险存在：美国和中国冲突

    瑞.达利欧的分析框架¶

    1.进化是宇宙中最大也是唯一永恒的力量
进化是走向进步的上行轨迹，而进步源于适应和学习

2.朝着一个方向的走势中，潜藏着导致反向走势的要素
这就是周期
人类的本性例如恐惧，贪婪，嫉妒和其他基本情感始终没有改变
它们是驱动周期发展的重要因素

3.进化和周期共同构成螺旋式上行轨迹
进化是一种相对平缓而稳定的进步
而周期则是来回波动的，像钟摆一样周而复始

4.人类的生产率是驱动世界总财富，权力和生活水平逐步提高的最重要的力量
不同群体的生产率增速有差异，其原因是不同的教育质量，创造力，职业道德以及将想法转化为产出的经济体制
成功的体系一直是受过良好教育的人们文明相处，提出创新想法，透过资本市场获得资金，将创新转化为生产和资源分配，从中获利而得到回报
最广泛地使用人才，并根据个人业绩而不是特权来赋予公民责任的社会最可能取得可持续性成功

5.改变人类向上进步命运的三大周期
债务/资本市场周期，内部秩序和混乱周期，外部秩序和混乱周期
世界秩序会按照一定规律发生周期性的变化
主要表现为世界强权国家的兴衰更替，也就是财富和权力的转移

6.8个主要实力指标来衡量各个经济体在兴衰过程中财富和权力的变化
教育，创新和技术，成本竞争力，军事实力，贸易，经济产出，市场与金融中心地位，以及储备货币地位
这8个指标，加上三大周期和其他实力指标，最终形成关于一个国家实力的全部宏观因子—"18个因素"

    瑞.达利欧对未来预测¶

    # 未来10年世界
人口将增长10%-15%
人均产出将增长20%左右
人均财富将增长30%
平均寿命将延长7.5%

    施一公 - 破局的关键在于创新能力¶

    # 创新是把事情做成的方法
让从不同角度看问题的人把不同的东西结合起来
创造有利的环境和共同的利益

# 不同类型的教育
1.技能教育：获取知识的教育
2.培养原创思维的教育

# 人生三个阶段
1.第一阶段是学习
我们依靠别人
2.第二阶段是聚焦于工作
我们成为身边人的依靠，试图获得成功
3.人生最后一个阶段
把自己认为有价值的东西传递给下一代，超越自我

# 自然进化过程
先照顾好自己，当不再需要为自己积累更多，有帮助他人的能力时，再去行动

# 行动方式
1.用慈善方式进行捐赠帮助
2.通过传递自身学到的知识和拥有的原则

# 个人如何应对变化中的世界
1.态度：把生活看成一场冒险，接受人生会有起起落落的事实，所有这些起起落落都只是现实的反映
2.保持好奇心和冒险精神，才能应对变化的世界
3.保持冷静，保持对身边的事物变化的敏锐度，做好自己正在做的事，便足以应对变化中的世界
