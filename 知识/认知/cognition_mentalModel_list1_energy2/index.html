    记录¶

    需要先梳理一下昨天和娃的争纷

不知道为什么老大现在要上厕所不关门
只有一碰门就急眼
另外还习惯边上厕所边看书
(这点大人要改，总拿着手机，不好的榜样)

# 从情绪上
"顶嘴"：顶嘴时不回应，避免权力之争
"认同感受"：虽然不了解不可以关门这事，应该先接受再了解背后的原因
"积极的暂停"：起冲突时，冷静下来，先让双方感觉好起来
"倾听"：平静后，避免说教，听娃说完
"无言的行动"：厕所看书不好，温柔而坚定的把书拿走，态度要平和，娃知道厕所看书是不好的，避免重复说教
"看着眼睛说话"：使说话温柔

# 去做
"纠正前先连接"：说我爱你，但是定规矩，大家都不在上厕所时做其他；制定时间
"控制自己的行为"：再生气，也别大声说教
"鼓励/表扬"：平常多找到地方表扬，鼓励而不是惩罚
"赋予孩子自主权"：如果坚持不关门，就尊重
"转移注意力&引导"，"关注于解决方案"：说话内容避免陷入情绪的宣泄中，"让娃平静下来，不再看书，专心上厕所"是目标，说什么能实行就说，否则沉默更佳。

# 学以致用，万里长征

    继续学习精力管理¶

    积极正面的情绪¶

    # 意义
心理学有大量的证据证明，情绪对人的记忆力、认知力、决策力都有影响
产出最高、创造力最强的时候不是处于波谷，而是波峰-正面情绪高涨时
积极正面的情绪是精力输出的保障

    控制情绪，缓解焦虑¶

    # 三条定律
1.脑子在一个时段只能主要存在一种情绪
2.相比正面情绪来说，人脑更容易产生负面情绪
3.人可以通过自主的训练来控制情绪
    训练¶
    # 激发正面情绪
- 热启动（Priming）
练习呼吸，感受心跳，回忆感恩的事，想值得改善和庆祝的事，想三个目标

# 缓解焦虑情绪
- 焦虑
焦虑的体验： =~ 担心或害怕；是对危险和不确定性的一种正常反应
放松呼吸法：冥想的一种基本技巧
情绪标签法：和放松呼吸法一起做

# 把焦虑的事情写下来并且列出相应的对策
过度专注，钻牛角尖 -> 转移注意力来缓解
找不到解决方法，担心，焦虑泛化 -> ZTD

# 感恩心态

# 运动

    放松呼吸法，情绪标签法

    随时可以聚焦的注意力¶

    # 意义
注意力让精力有效的输出，创造有效的结果
注意力是能够自主控制最重要的资源

    如何聚焦¶

    # 大脑输出方式
1.事务性或叫应付性输出
2.系统性/创造性输出

# 三个方法
1.SMART目标
2.记录碎片信息 + 一分钟念头记录法《零秒思考》
3.最好的时间给最重要的事

# 仪式感->快速进入心流四步法 (贾行家老师)
1.倒计时，eg. 打开番茄钟，开声音
2.马上上手，eg. 写出第一句话比写得好重要
3.保持惯性，eg. 有错误不顺畅，不停顿，回头再修改
4.选择中断点 -> 回来时，不必重复前3步进入心流

    明确的意义感¶

    # 意义感
是人活着的最高追求
是驱动做事的底层逻辑
是人生的操作系统
是精力的最终源泉

# 人生的意义在于用自己所长服务他人
1.我擅长做什么
能给他人当老师的程度
2.我做这事服务谁
3.他从服务中能得到什么
4.我的服务能够让他有什么不同

# 使命：MPS三圈交集
Meaning：意义
Pleasure：快乐
Strength：优势

使命是意义感的来源
找到意义感，如同航海看到灯塔，开车看到路标
事业有方向，遇事不纠结
使生命迸发出巨大能量，产生持久的精力

    行动清单¶

    # 晨间7事
1.醒时，在床上做简单活动(搓脸梳头拉伸)，回忆梦，在潜意识里挖宝
2.整理床铺，穿戴整齐，形成心理暗示："我马上完成了一件事，我可以成就一件事"
3.喝一大杯温水
4.用不常用的那只手刷牙
5.做情绪热启动练习
6.标记今天最重要的三个目标，并想象完成后的快乐
6.健康，营养，低热量的早餐

# 日间安排
1.今日三只青蛙，策划思考
2.按照优先级处理任务，番茄钟
3.工作间隔，简单运动，整理todo

# 睡前n事
1.几分钟冥想
2.回顾三个目标是否完成
3.看时间记录，是否重要的时间花在重要的事上
4.看书

    如同肌肉可以在健身房练出来，精力也可以通过训练不断提升

    习惯不是短时间养成，从自己最comfortable的地方开始

    安定从容，专注并享受：

    1.运动后的快感

    2.美食的味道

    3.完成一件事后的成就感，哪怕只是折被子的简单小事

    4.在嘈杂的环境，专注呼吸，内心平静

    冥想¶

    # 得到童慧琦的正念冥想

# 正念 (mindfulness)

# 正念练习
有关如何使用和安放注意力的一种方法
你的身体就是你的实验室
通过练习，提升专注力，激发内在的潜能，并且找到和压力相处的方式

# 正念三要素
念，上面是今，下面是心，今心为念
一颗处当下的心就是一个念的状态
1.有意关注
把注意力导向某一个对象或者目标
2.非评判
对评判有所觉察：喜欢，不喜欢，既喜欢又不喜欢都是评判
不被评判牵着鼻子走
3.当下
"当下"是身心所体验到的一切
- 内在体验：脑子的一个念头，身体的感觉
- 外在现象：五感

正念练习的态度，学会打破自己对结果的期待
