    继续学习沟通思维¶

    非暴力沟通¶

    马歇尔·卢森堡博士 Marshall Rosenberg (1934—2015)
师从人本主义心理学之父卡尔·罗杰斯，美国威斯康星大学临床心理学博士

# 暴力沟通
言语上的指责
嘲讽
否定
说教
任意打断
拒不回应
随意出口的评价和结论

# 负面情绪
愤怒，沮丧，焦虑
冷漠，隔膜，敌视
自私自利，贪婪，
偏见，怀疑

# 积极情绪
尊重，理解，欣赏
感激，慈悲，友情

    让爱融入生活¶

    # 非暴力沟通四要素
1．观察 
2．感受 
3．需要 
4．请求

专注于彼此的观察，感受，需要和请求
鼓励倾听，培育尊重与爱
乐于互助

    是什么蒙蔽了爱¶

    # 道德评判
对他人的评价反映了自己的需要和价值观
忽视彼此的感受与需要，将冲突归咎于对方

# 进行比较
"比较"也是一种评判
"比较"蒙蔽了对人对己的爱意

# 回避责任
用负责任的语言代替回避责任的语言
无法强迫他人按自己的期待生活

# 强人所难
认为"某人应当受到惩罚"使难以体会到心中的爱

    区分观察和评论¶

    - 不区分观察和评论，人们将倾向于听到批评，并反驳
- 负面标签的消极影响很明显
- 正面或中性的标签也会妨碍全面了解一个人
- 不带评论的观察是人类智力的最高形式

# 注意词语使用在表达观察结果中还是评论
"每次"，"曾"，"总是"，"从不"，"经常"，"很少"

非暴力沟通是动态的语言，不主张绝对化的结论
提倡在特定的时间和情境中进行观察，并清楚地描述观察结果

    体会和表达感受¶

    # 建立表达感受的词汇表
1. 得到满足时的感受
兴奋 喜悦 欣喜 甜蜜 精力充沛 兴高采烈 感激 感动 乐观 自信 振作 振奋 开心 高兴 快乐 愉快 幸福 陶醉 满足 欣慰 心旷神怡 喜出望外 平静 自在 舒适 放松 踏实 安全 温暖 放心 无忧无虑

2. 没有得到满足时的感受
害怕 担心 焦虑 忧虑 着急 紧张 心神不宁 心烦意乱 忧伤 沮丧 灰心 气馁 泄气 绝望 伤感 凄凉 悲伤 恼怒 愤怒 烦恼 苦恼 生气 厌烦 不满 不快 不耐烦 不高兴 震惊 失望 困惑 茫然 寂寞 孤独 郁闷 难过 悲观 沉重 麻木 精疲力尽 委靡不振 疲惫不堪 昏昏欲睡 无精打采 尴尬 惭愧 内疚 妒忌 遗憾 不舒服

# 更清楚地表达感受，使沟通更为顺畅

# 在表达感受时，示弱有助于解决冲突

# 区分表达
具体感受的词语
陈述想法，评论以及观点的词语

    感受的根源¶

    # 听到不中听的话，四种选择
1. 责备自己
2. 指责他人
3. 体会自己的感受和需要
4. 体会他人的感受和需要

# 他人的行为会刺激我们，但并不是我们感受的根源

# 忽视感受与自身关系的表达
1. 只提及相关的事情
2. 只提及他人的行为
3. 指责他人

# 表达感受与自身关系的句式
我（感到）。。。因为我。。。
    生命健康成长的要素¶
    通过批评来提出主张 -> 反应通常是申辩或反击
直接说出需要 -> 较可能作出积极的回应

1. 自由选择（Autonomy）
选择梦想/目标/方向  Choosing dreams/goals/values
自由制定计划来实现这些梦想、目标和方向 Choosing plans for fulfilling one’s dreams，goals，values)

庆祝生命的创造力以及梦想的实现 Celebrate the creation of life and dreams fulfilled
纪念人生的失落：亲人的去世或梦想的破灭等（表达悲伤）Celebrate losses: loved ones，dreams，etc. mourning

    个人成长三个阶段¶
    1. 情感的奴隶
认为自己有义务使他人快乐

2. 面目可憎时期
拒绝考虑他人的感受和需要

3. 生活的主人
意识到，虽然对自己的意愿，感受和行动负有完全的责任，但无法为他人负责。与此同时，还认识到，无法牺牲他人来满足自己的需要

    请求帮助¶

    # 提出具体的请求
请求越具体越好
意思含糊不清，别人难了解到底想要什么

# 明确谈话的目的
对自己的认识越深刻，表达越清楚

# 请求反馈
当对方给予反馈，表达感激
对方不愿反馈，倾听他的感受和需要

# 了解他人的反应
对方此时此刻的感受
对方正在想什么
对方是否接受请求

# 在集体讨论时提出请求
说清楚希望得到怎样的反馈
清楚发言的目的

# 请求与命令
- 区分命令和请求
命令：
请求没有得到满足，提出请求的人批评和指责
利用对方的内疚来达到目的

# 目的
帮助在诚实和倾听的基础上与人联系
改变和行动是出于对生命的爱
当相信请求是因看重彼此的感情，并希望兼顾双方的需要，就会相信所表达的是愿望

    用全身心倾听¶

    倾听他人意味着，放下已有的想法和判断，一心一意地体会他人
分析妨碍了倾听

倾听有助于对他人的理解和接纳
不论别人说什么，只听到对方此时此刻的观察，感受，需要，和请求
对方得到充分的倾听，将体会到气氛变得轻松，会停止谈话
打断别人比假装在听，更符合对方的愿望

# 越是留意自己内心的声音，就越能够听到别人的声音
不愿意示弱的时候往往是因为担心失去控制想显得强硬
一心分析自己或对方的过错，会认为自己被贬低了

当认为自己受到指责，意识那些话是他人表达需要和请求的方式
当对方说"不"的时候，体会他的感受和需要，会发现无法答应请求的原因

# 当无法倾听他人时
1.体会自己的感受和需要
2.大声地提出请求
3.换一个环境

    倾听帮助人们治愈心灵的创伤
一旦有人倾听，看起来无法解决的问题会变得清晰

    爱自己¶

    非暴力沟通最重要的应用也许在于培养对自己的爱

改变是出于对生命的爱，而不是出于羞愧或内疚等负面影响的心理
自责是尚未满足的需要的可悲表达

# 直面人生的苦难
在遇到挫折时，充分体会人生的悲哀和内心的渴望
感到遗憾是难免的
从经历中学习，而无须责备自己

# 自我宽恕为个人的学习和成长创造条件
评价自己的行为时，专注于尚未满足的需要
一旦专注于尚未满足的需要，就会考虑如何满足它
不再依赖羞愧，内疚，恼怒或沮丧的心理来寻求改变
让爱主导学习和成长

# 爱惜自己的关键
同时包容两个不同的我
1.包容采取那种行为的我
2.包容对过去的某种行为感到后悔的我

# 用"选择做"代替"不得不"
不论选择做什么，了解为什么
1.列出在生活中觉得没意思又不得不做的事
2.标出哪些是自己选择的
3.一旦承认某一行为是自己的选择，声明为什么
我选择做。。。是因为我想要。。。

# 主动根据需要和价值观来选择生活
深入理解行为的动机
行为不是为了履行职责，获得回报
"选择做"，将生活变得和谐并充满欢乐

    充分表达愤怒¶

    不忽视或压抑愤怒
深入了解愤怒，充分表达内心的渴望

生气的原因在于自己的想法 — 对他人的评判和指责
愤怒是思维方式造成的，核心是尚未满足的需要

# 将愤怒转变为服务需要的情感
意识到自己的需要

# 表达愤怒的步骤
1．停下来，呼吸
2．留意自己的指责
3．体会自己的需要
4．表达感受和尚未满足的需要

# 越是能够倾听他人，也越有机会被倾听

# 先倾听他人
如果对方还处于某种情绪中，很难静下心来体会我们的感受和需要
用心倾听对方，并表达我们的理解
在得到倾听和理解之后，对方一般也就会开始留意我们的感受和需要

# 只要人们认为自己受到了指责，就很难体会到别人的痛苦

    善用强制力¶

    使用防卫性的强制力，是为了保护自己或他人，而不是惩罚

# 二个问题替代惩罚的念头
如果不喜欢他现在的行为，那希望他怎么做
希望他基于怎样的原因去做想要他做的事情

    生活的热情¶

    倾听和表达自己的需要并不容易
文化倾向于把个人需要看作是消极的
一个人公开表达自己的需要，可能被看作是自私

沮丧是一个人处于激烈的内心冲突之中，无所适从

以苛刻的态度对人对己，心情好不到哪里
不试图分析自己或他人有什么毛病
用心去了解彼此的需要

体会他人的感受和需要也可以释放压力

发现自己心底深处的愿望，采取积极的行动 -> 生活的热情

    表达感激¶

    注意赞扬的动机，不把自己放在了裁判的位置
赞扬并不总是有助于深化彼此的联系

# 表达感激的方式
1．对方做了什么事情使我们的生活得到了改善
2．我们有哪些需要得到了满足
3．我们的心情怎么样

一件值得做的事情即使做得不怎么样也是值得感谢
