    记：孩子 - 正面管教5¶

    记录¶

    昨天去滑雪，回来的时候说再滑一次，直接滑到停车场。
老大不愿意了，有情绪，说摔了累了，想直接回家。
在雪场，我们一个山上，一个山下，
互相能看到，僵持了约半个小时，
最终还是按计划，再滑了一次去停车场。

我没忍住说道理，
当时觉得这孩子怎么每次都要整点事，
不理解每一次一件快乐的事情，
怎么都能变得这么不愉快结束。

我也认为我的处理能力特别糟糕，
就和幽默感一样，
没有，
就没能力说出那些化解尴尬，转换气氛的话。

哎，记了这么多正面管教的方法，
好好回顾一下都有哪些点可以用。

    昨天原本可以这样的¶

    当老大表示不愿意继续滑时，
"倾听"：摔倒了，手冷，累了
"给与关注"：摸摸腿，手，"拥抱"
说"我注意到"：复述摔倒的过程
"认同感受"：确实是挺累了

当"顶嘴"时，保持沉默，"积极暂停"

"无言的行动"：这个倒是算采用了，直接先滑下山 (捂脸)

在车上，应该"近距离倾听"，不判断不防御不解释

回家后，可用"修复错误三部曲"，今天来试试：承认，和好，找到下次更好的解决方式

    继续学习¶

    激励¶

    # 很多错误都是以激励的名义犯下的
1.我们从哪里得到这样疯狂的想法，当孩子感觉糟糕时会做的更好（惩罚）
2.孩子们感觉好时才会做的更好（鼓励）
3.和孩子一起寻找问题的解决办法来激发孩子的动力

    转移注意力&引导¶

    # 取代说"不要..."，而是引导到"做..."
1.把孩子带离电视，引导说，"你可以玩积木"
2.转移注意力，变成"挠痒痒怪兽"
3.手拉手把孩子带到另一项活动
4.给孩子一个计时器，让他告诉你什么时候可以离开公园

    控制你自己的行为¶

    # 榜样是最好的老师
1.当你无法控制自己的行为时，还希望孩子能控制他们的行为吗
2.创建你自己的特别暂停区，当你需要使用时告诉孩子
3.如果你不能离开冲突现场，数到10，或者深呼吸
4.当你犯错误时，对孩子道歉

    选择轮¶

    # 教孩子使用选择轮是解决问题的一种办法
1.（和孩子一起）brainstorm，列出每日冲突或问题的可能解决办法清单
2.在一个圆形图表上的每部分写下一个解决方法，让孩子画图或标记
3.有冲突时，建议孩子使用选择轮找到一个解决办法

    愤怒选择轮¶

    1.教孩子有情绪是正常的，但他们的所作所为不一定总是对的
2.在孩子平静的时候，展示愤怒选择轮，看看表达愤怒的其它尊重性的替代方法
3.孩子生气时，认同他/她的感受，提供选择："现在有什么能帮到你:积极暂停，还是选择轮"

    看着他的眼睛¶

    # 坐在沙发上，对着房间另一端的孩子叫喊是不尊重的（也不管用）
1.停下你正在做的事，站起来，走近孩子，看着他/她的眼睛
2.你会注意到，当你作出尊重的努力，看着孩子的眼睛时，你说话会更温柔

    一个词¶

    # 避免说教和挑剔。使用一个词作为友善的提醒
1."毛巾"(指掉在地板上的毛巾)
2."狗"(没喂狗的时)
3."盘子"(饭后帮收拾时)
4."睡觉时间"
如果事先共同达成约定，通常说一个词就够了

    赋予孩子自主权¶

    # 与孩子分享控制，以便他们发展出掌控生活所需的技能
1.教导生活技能
2.着重于共同解决问题
3.信任孩子
4.放下(一步步地)
5.增长自我认知，问："你感觉如何？你怎么想？这件事对你的生活有什么影响"

    避免娇纵¶

    # 家长的一个错误是：以爱的名义娇纵
1.娇纵造成孩子的懦弱，因为它让孩子形成这样的信念：其他人应该为他们做好所有的事情
2.你能够给孩子的最好礼物之一，允许他们发展“我能行”的信念
3.当孩子学习到自己能够在生活中的坎坷中生存时，就有了能力感

    破解密码¶

    # 使用任何正面管教书中的"误解目的表"
1.选择一个问题行为
2.辨别你的感受，以及你是如何应对的
3.辨别当你告诉孩子停止时，孩子的反应
4.使用"误解目的表"，辨别孩子行为背后可能的信念
5.尝试"误解目的表"最后一栏的建议，鼓励行为的改变

    少用逻辑后果¶

    # 逻辑后果很多情况下是欲盖弥彰的惩罚
1.尽可能少用逻辑后果，相反，关注于解决问题
2.合适的时候，遵循逻辑后果的3R1H
  a.相关(Related)
  b.尊重(Respectful)
  c.合理(Reasonable)
  d.有帮助(Helpful)

    细小步骤¶

    # 把任务细化，让孩子体验成功;教技能，但不要为他们做;当孩子能完成细小步骤时，就会放弃"我不行"的信念
eg. eg. 练习新曲子
1.把谱子认一遍
2.唱打节拍
3.先练一只手
4.遇到困难的小节，只反复练习它，最少三遍正确，再前后连起来

    鼓励¶

    # "孩子需要鼓励，就像植物需要水" - 鲁道夫·德雷克斯
1.一个行为不当的孩子是一个丧失信心的孩子。当孩子感受到鼓励时，不当行为就消失了
2.通过"纠正前先连接"来鼓励孩子
3.制作卡片帮助孩子感到鼓励，发展有价值的社会和生活技能，帮助他们感到有能力

    鼓励/表扬¶

    1.教孩子自主，而不是依赖于他人
2.鼓励引发自我评价
3.表扬让孩子成为"寻求认可迷"
eg. 表扬："我为你骄傲。这是你的奖品"
鼓励："你努力了，你一定为自己感到骄傲"

    共赢的合作¶

    # 当父母理解并尊重孩子的观点时，孩子感到深受鼓舞
1.对孩子的想法和感受表示理解
2.共情但不怜悯
3.分享你有类似的感受或行为的情形
4.分享你的想法和感受(只有当孩子感到被倾听时才会听你的)
5.关注于共同解决问题
